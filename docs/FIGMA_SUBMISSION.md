### Description

```text
Find the right Nigerian financial logo without leaving Figma.

awalogo is an open-source catalog of logos from Nigeria's financial ecosystem. Search more than 200 available assets from banks, fintechs, payment providers, insurers, investment firms, pension companies, regulators, and related institutions.

WHAT YOU CAN DO

- Search by company name, alias, category, licence, or regulator.
- Filter by one or more financial institution categories.
- Choose SVG, PNG, or WebP when those formats are available.
- Browse alternate logo variations where available.
- Set an exact width or height with an optional locked aspect ratio.
- Insert editable SVG vectors or raster image layers.
- Insert a logo directly into a selected frame.
- Clip a logo to a selected shape.
- Replace a logo previously inserted by awalogo while preserving its placement and size.
- Copy supported assets for use elsewhere in your design workflow.
- Use system, light, or dark color mode.

SOURCE-AWARE AND OPEN SOURCE

Each available logo includes institution metadata, source status, and an official website where available. The catalog is maintained in public, and companies and community contributors can request a missing logo, submit current artwork, or report a correction.

OFFLINE BY DESIGN

The catalog and logo files are bundled with the plugin. awalogo does not require an account, payment, or runtime network connection. Website and GitHub links open outside the plugin only when you choowhat se them.

TRADEMARK NOTICE

Logo artwork and company names are trademarks of their respective owners. They are not relicensed by awalogo. Check each brand's current guidelines before use. awalogo is an independent open-source project and is not affiliated with or endorsed by the institutions represented in the catalog.
```

## Data Security Disclosure

Use the following answers in Figma's data security form.

### Summary

```text
awalogo does not collect, store, sell, share, or transmit personal data. The logo catalog and assets are bundled with the plugin and work without runtime network access.
```

### Network access

```text
No access to network.
```

The manifest declares `allowedDomains: ["none"]`.

### Accounts and payments

```text
No account, sign-in, subscription, or payment is required.
```

### User data collection

```text
The plugin does not collect analytics, identifiers, document contents, personal information, or usage history.
```

### Local storage

```text
The plugin may store the user's color-mode preference locally in the plugin UI. This preference is not transmitted and does not identify the user.
```

### Figma document access

```text
The plugin reads the current selection only when needed to place, replace, or clip a logo. It writes a logo layer only after the user chooses Insert. It does not upload or transmit file contents.
```

### Requests and submissions

```text
Logo request and company submission forms create a prefilled GitHub issue URL. The plugin does not submit or store the form contents. The user reviews and submits the issue on GitHub, where GitHub's terms and privacy policy apply.
```

### Advertising and tracking

```text
The plugin contains no advertising, analytics, behavioral tracking, or third-party tracking SDKs.
```

## Reviewer Notes

Paste this into any review-notes or additional-information field:

```text
awalogo is a free, offline-first logo browser for Nigerian financial institutions.

No account or payment is required. The plugin makes no runtime network requests and declares allowedDomains: ["none"]. All catalog data and logo assets are bundled locally.

Suggested review flow:
1. Run the plugin in a Figma Design file.
2. Search for "Kuda" or "Sycamore" and open a result.
3. Choose an available format or logo variation.
4. Set a width or height and toggle the aspect-ratio lock.
5. Select Insert to add the logo to the canvas.
6. Select an unlocked frame and insert another logo to place it inside the frame.
7. Select an unlocked shape and insert a logo to test clipping.
8. Select a logo previously inserted by awalogo, choose another logo, and insert it to test replacement.
9. Switch between system, light, and dark color modes.

SVG assets are inserted as editable vectors. PNG and WebP assets are inserted as raster layers. Available formats vary by institution.

External links to awalogo.com and GitHub open only after a user selects them. Request and submission forms prepare a GitHub issue for the user to review; the plugin does not submit those forms itself.

Logo artwork and company names remain trademarks of their respective owners. The open-source project maintains source metadata, validation tooling, corrections, and a public takedown path.
```

## Preflight Checklist

- Enable two-factor authentication on the publishing Figma account.
- Build with `pnpm build:plugin`.
- Import `apps/figma-plugin/manifest.json` in Figma Desktop.
- Confirm `apps/figma-plugin/figma-dist/main.js` loads.
- Confirm `apps/figma-plugin/figma-dist/index.html` renders offline.
- Test search, filtering, variations, sizing, copy, and insertion.
- Test frame insertion, shape clipping, and selected-logo replacement.
- Test system, light, and dark modes.
- Test empty search and missing-logo request flows.
- Upload the 128 x 128 icon.
- Upload the 1920 x 1080 thumbnail.
- Verify screenshots accurately match the submitted build.
- Confirm the support contact is monitored.
- Confirm the repository URL is `https://github.com/adeyemimayokun/awalogo`.
- Confirm the plugin is submitted as free.
- Review Figma's current plugin review and Community guidelines before submitting.

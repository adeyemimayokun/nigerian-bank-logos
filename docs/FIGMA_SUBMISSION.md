# Figma Community Submission Notes

## Plugin Name

Nigerian Bank Logos

## Short Description

Search and insert verified Nigerian bank and fintech SVG logos.

## Long Description

Nigerian Bank Logos helps designers insert bank, fintech, and payment provider
logos into Figma from a source-aware open catalog. The plugin works offline and
ships with verified SVG files from official web properties.

Logo trademarks belong to their respective owners. The open source project
maintains metadata, validation, and contribution tooling.

## Privacy

The plugin does not make network requests in v1 and declares no allowed network
domains.

## Review Notes

- Import `apps/figma-plugin/manifest.json`.
- Build output is generated with `pnpm build:plugin`.
- Main file: `apps/figma-plugin/dist/main.js`.
- UI file: `apps/figma-plugin/dist/index.html`.

# Introducing awalogo

I kept running into the same problem while designing products at [Sycamore](https://sycamore.ng).

Whenever we needed the logo of a bank, fintech, or payment provider, the design team could search the web and add it to a prototype. But once the product moved into development, that workaround became inefficient.

Developers needed reliable, reusable, and optimized assets—not another folder filled with duplicate, outdated, or poorly formatted logo files.

I searched for an existing solution, but most logo libraries had limited coverage of Nigerian institutions. Some were no longer regularly updated, while others were missing newer banks and fintechs.

So I decided to build **awalogo**.

The name is a familiar Nigerian expression meaning **“our logo.”** It represents what I want the project to become: a shared, community-driven resource for designers and developers.

I started with Nigeria’s financial ecosystem—banks, fintechs, payment providers, regulators, insurers, and other financial institutions.

Bringing the idea to life involved:

- Compiling a structured list of institutions.
- Sourcing artwork from official websites and brand-owned materials.
- Reviewing assets instead of accepting arbitrary web images.
- Preparing available SVG, PNG, and WebP formats.
- Building a searchable public catalog.
- Creating an offline-first Figma plugin.
- Establishing contribution and review workflows so the library can grow responsibly.

Today, awalogo contains **119 reviewed logos sourced from official properties**, and the collection is still growing.

The project is open source and free to use. The supporting code and tools are open to the community, while every logo remains a trademark of its respective owner.

My long-term goal is to expand beyond financial services into other Nigerian industries—and eventually include institutions from across Africa and the rest of the world.

If you are a designer, developer, or contributor, I would love for you to explore the project, use it, and help improve it.

Visit **[awalogo.com](https://awalogo.com)**.

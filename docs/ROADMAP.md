# Roadmap

## 0.1 Foundation

- Monorepo scaffold.
- Figma plugin MVP.
- Verified seed SVG catalog.
- Validation and CI.
- Contribution and trademark docs.

## 0.2 Bank Expansion

- Add verified SVGs for commercial banks where official vector sources exist.
- Add deprecated/renamed institution handling.
- Generate README gallery from catalog metadata.

## 0.3 Developer Packages

- Publish `@awalogo/core`.
- Add raw SVG string exports.
- Add React helpers.
- Add plain HTML and Next.js examples.

## 0.4 Design Ecosystem

- Add Iconify-compatible export.
- Add Figma plugin screenshots and polish pass.
- Add optional Figma library generation workflow.

# Nigerian Bank & Fintech Logos PRD

## Summary

Create an open source logo library for Nigerian banks and fintechs, starting
with a Figma plugin and using a structured logo data package underneath it.

## Users

- Designers who want searchable, insertable SVG logos in Figma.
- Developers who want clean, optimized SVG assets.
- Contributors who can add or correct verified assets.

## v1 Scope

- Searchable Figma plugin.
- Offline bundled SVG catalog.
- Logo metadata with name, category, aliases, source, status, and SVG path.
- Validation for metadata, duplicate slugs, missing files, and unsafe SVG.
- Contribution workflow and trademark policy.

## Later Scope

- Published npm package.
- React component package.
- Iconify-compatible export.
- Generated documentation gallery.
- Larger verified catalog across commercial banks, MFBs, fintechs, and payment
  providers.
